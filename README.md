# Claude Demo — Enron Forensic Dataset

A workshop dataset built around the Enron scandal, combining synthetic transaction data with real-world offshore leak records and email corpus entity extraction.

## Data Generation

### `data/structured/transactions.csv` & `transactions_facilitator.csv`
Generated by `scripts/generate_transactions.py` using a fixed random seed (default: 42).
Produces 2,000 synthetic Enron-era wire transfers embedding four forensic patterns:
- **Structuring** — 14 transfers to Chewco just below the $10k reporting threshold
- **Round-trips** — Enron → LJM Cayman → JEDI → Enron (debt disguised as equity)
- **Shell payments** — recurring consulting fees to Cayman/BVI entities
- **Quarter-end spikes** — large transfers on the last days of each quarter

The workshop version (`transactions.csv`) omits the `flag_type` column; the facilitator version includes it as an answer key.

### `data/structured/accounts.json`
Also produced by `generate_transactions.py`. Contains metadata for all 27 accounts referenced in the transaction data: Enron Corp, 6 executives, 10 shell entities, and 10 legitimate counterparties.

### `data/structured/extracted_entities.csv`
Generated by `scripts/extract_enron_entities.py` against the [Enron email corpus](https://www.cs.cmu.edu/~enron/) (517,401 emails). Scans for known executive names and special purpose entity (SPE) names, recording email frequency and sample context for 33 matched entities.

### `data/reference/icij_bridge.csv`
Generated by `scripts/build_icij_bridge.py` against the [ICIJ Offshore Leaks bulk database](https://offshoreleaks.icij.org/pages/database). Cross-references Enron entity names against 814,344 ICIJ nodes, yielding 371 matches across 12 leak datasets (Panama Papers, Paradise Papers, Pandora Papers, Bahamas Leaks, and others).

### `data/unstructured/emails/`
A curated selection of 100 real emails from the [Enron email corpus](https://www.cs.cmu.edu/~enron/), filtered to include only messages mentioning key SPEs and individuals: LJM, Chewco, JEDI, Southampton, Fastow, Raptor, Whitewing, Osprey, Marlin, and Timberwolf. Up to 15 emails per keyword, organised by mailbox owner (blair, corman, cuilla, dasovich, fastow, harris, lay, martin, may, mckay, nemec, presto, ring, skilling, taylor, whalley, benson).

## Usage

```bash
pip install -r requirements.txt

# Regenerate synthetic transactions
python scripts/generate_transactions.py

# Extract entities from Enron email corpus (requires local copy of maildir)
python scripts/extract_enron_entities.py --enron-dir /path/to/maildir

# Build ICIJ bridge (requires ICIJ bulk CSV download)
python scripts/build_icij_bridge.py --icij-nodes /path/to/nodes-entities.csv
```
